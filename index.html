About Me

I've always had an interest in programming. It started back in sixth grade, when I had a dream about making a game. Upon waking up from that dream, I decided it was time to learn how to program. Ten years later, through self-teaching and some programming classes, I'm here.

The in-depth:

I've worked in the general vicinity of the C languages most of my career, with Python being probably my biggest outlier to that. JavaScript is currently my favourite, as I've found it easy to whip up simple test projects and tinkers with minimal overhead involved, but I can still appreciate the other languages such as Java and C# for their power and efficiency.

Some of the languages I've picked up along the way, in order of most experience:

  • - JavaScript - Self-taught, 2 years
  • - Visual Basic - My first language, 3 years
  • - TypeScript - Learned at Eleven Fifty, 1 year
  • - Java - Self-taught / schooling, 2 years
  • - C# - Learned at Eleven Fifty, 1 year
  • - HTML/CSS - Self-taught / schooling, 3 years
  • - C++ - High school / college, 2 years
  • - Python - Self-taught / college, 1 year

Some personal values:


Honesty and open communication. Without these, no relationship, whether business, personal, romantic, or any other can succeed well.
